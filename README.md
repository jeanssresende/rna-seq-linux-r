## Course overview

This repository contains the teaching materials, scripts, datasets and exercises used in the RNA-Seq Analysis course.

Topics covered include:

- Linux command line
- Shell scripting
- FASTQ file structure
- Quality control (FastQC & MultiQC)
- Read trimming (fastp)
- Transcript quantification (Salmon)
- Importing counts into R
- Differential expression analysis (DESeq2)
- Data visualization
- Functional enrichment analysis
